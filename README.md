# Alterian Email Manager API WebForm

Alterian Email Manager API ASP.Net Web Form example.

This example shows how to use Alterian Email Manager C# client library in ASP.Net Web Form. 

## Steps
1. Install Alterian Email Manager API client library nuget package in Web Form project
```
PM> Install-Package AlterianEmailManagerAPI
```
2. Add client side logic utilzing the library
